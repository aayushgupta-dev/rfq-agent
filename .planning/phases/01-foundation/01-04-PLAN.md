_find_latest(prompt_id: str, base_dir: Path) -> Path
  glob f"{prompt_id}.v*.md" in base_dir
  parse filename suffix .vN.md -> int version
  return the highest-version file path
  raise KeyError(f"no prompt '{prompt_id}'") if none found

load(prompt_id: str, base_dir: Path = _DIR) -> frontmatter.Post
  validate prompt_id against re.fullmatch(r'^[a-z0-9-]+$', prompt_id); raise ValueError if invalid
  call _find_latest(prompt_id, base_dir) to get the file path
  return frontmatter.load(file_path)
  import: import frontmatter  (the python-frontmatter package, installed in Plan 01)
  _DIR = Path(__file__).resolve().parent  (the prompts directory)

Frontmatter keys per stub (D-11): id, version, intent, model_tier, failure_handling
model_tier values map to the LLM factory tiers: "reasoning" | "cheap"
