From services/ai/schemas/envelope.py (the gate's I/O contract — read this file first):

class FlagStatus(StrEnum):
    present = "present"; missing = "missing"; unclear = "unclear"
    conflicting = "conflicting"; unsupported = "unsupported"

class Evidence(BaseModel):
    snippet: str; char_start: int; char_end: int; source_id: str

class ConflictingValue(BaseModel, Generic[T]):  # noqa: UP046
    value: T | None = None
    evidence: list[Evidence] = pydantic_Field(default_factory=list)

class Field(BaseModel, Generic[T]):  # noqa: UP046
    status: FlagStatus; value: T | None = None
    evidence: list[Evidence] = pydantic_Field(default_factory=list)
    values: list[ConflictingValue[T]] | None = None

Key semantic rule: status=unsupported → model_validator enforces value=None, evidence=[].
Construct unsupported downgrade as: EnvelopeField(status=FlagStatus.unsupported)
The validator sets value=None and evidence=[] automatically.

Offset recompute: model_copy(update={"evidence": new_evidence_list}) with
Evidence(snippet=ev.snippet, char_start=recomputed_start, char_end=recomputed_end,
source_id=ev.source_id) — recomputed offsets replace original ones.
