## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → /extract/file-text | File bytes cross this boundary |
| browser → /input/raw-text | Pasted text crosses this boundary |
| Route handler → filesystem | /api/traces/[name] reads files from public/traces/ |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-04-A | Tampering | /api/traces/[name] filename param | mitigate | `name.replace(/[^a-zA-Z0-9_.-]/g, "")` sanitization before path.join; reads from public/traces/ only |
| T-05-04-B | DoS | File upload UI (client side) | accept | Server enforces 20 MB limit (T-05-02-B); client shows no extra guard (redundant) |
| T-05-04-C | Spoofing | LLM fabrication risk on Vendor Input | accept | Raw text is passed through to the extraction agent which enforces grounding; no client-side fact display on this screen |
</threat_model>
